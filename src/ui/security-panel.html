Enter credentials here, never in chat. Values are written directly to the credential store and never reach any model, log, or session file.

Heads up. If your browser asks to save these passwords, decline. The credential store on this machine handles persistence — your browser doesn't need a copy. Pasted values are auto-cleared from the clipboard after 30 seconds.
